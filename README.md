# Central da Mari

Minha central pessoal de tarefas, notas e OKRs — com a identidade visual da Serena Energia.

## Como usar

Acesse: `https://SEU-USUARIO.github.io/central-da-mari/`

## Funcionalidades

- **Board** — Kanban com 4 colunas (Foco, Próximas, Esperando, Feito)
- **Notas** — Anotações de reuniões e projetos vinculadas a tags
- **OKRs** — Objetivos e Key Results com barras de progresso
- **Performance** — Métricas mensais de tasks vs projetos por tag
- **Exportar/Importar** — Backup em JSON dos seus dados
- Tags customizáveis, drag-and-drop, lixeira com restauração
- Dark mode com grafismo Serena

## Dados

Os dados ficam salvos no `localStorage` do navegador. Use o botão **Exportar** para fazer backup.
