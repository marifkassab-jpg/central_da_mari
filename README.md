# Central da Mari

Minha central pessoal de tarefas, notas, OKRs e performance — com a identidade visual da Serena Energia.

## Como usar

Acesse: `https://marifkassab-jpg.github.io/central_da_mari/`

## Funcionalidades

### Board
- Kanban com 4 colunas (Foco da semana, Próximas, Esperando alguém, Feito)
- Drag-and-drop entre colunas
- Tags customizáveis por área (PBCS, Razão, Dados, Forecast, Budget + tags próprias)
- Classificação Task vs Projeto em cada card
- Data de prazo e data de criação automática
- Indicador de "inadimplência" — cards com +14 dias sem conclusão
- Lixeira com restauração
- Filtros por tag

### Calendário
- Vista mensal com navegação entre meses
- Cards posicionados por data de prazo, criação ou conclusão
- % de conclusão por dia
- Detalhe dos cards ao clicar no dia
- Filtros por tag

### Notas
- Anotações de reuniões e projetos
- Vínculo com tags
- Editor com título e corpo de texto

### OKRs
- Objetivos: Data Governance e Automação dos fluxos de FP&A
- Key Results com barras de progresso e sliders
- Possibilidade de adicionar novos KRs

### Performance
- Métricas mensais: total concluído, tasks vs projetos, mix %
- Breakdown por tag
- Gráfico de histórico mensal (tasks em cinza, projetos em coral)
- Seção "Inadimplentes" — cards antigos sem conclusão
- Lista de itens concluídos por mês
- Seletor de mês

### Dados e Backup
- Exportar todos os dados em JSON (botão ⬇ Exportar)
- Importar de um backup JSON (botão ⬆ Importar)
- Dados persistem no localStorage do navegador
- Botão Resetar para voltar ao estado inicial

## Stack
- HTML + CSS + JavaScript puro (zero dependências)
- Fonte: DM Sans (Google Fonts)
- Tema: dark mode com grafismo Serena (faixas diagonais)
- Hospedagem: GitHub Pages
